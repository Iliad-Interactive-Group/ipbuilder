# **App Name**: IPbuilderAI

## Core Features:

- Document Import: Upload a document (PDF or Word file) describing a company or product and import data.
- Form Fill: Use a form to collect data
- Copy Generation: Generate marketing copy variations tailored for websites, social media, blog posts, radio/TV scripts, podcast outlines, and billboards using an AI writing tool.
- Content Type Selection: User selects a content type (e.g., website copy, social media post, radio script).
- Keyword Input: Allow users to input additional keywords or refine instructions to include in copy generation.
- Copy Display: Display generated marketing copy.
- Copy Export: Allow users to export selected copy in common document formats (e.g., TXT, DOCX).

## Style Guidelines:

- Primary color: Soft, inviting purple (#B19CD9) to evoke creativity and innovation.
- Background color: Light grey (#F0F0F0), providing a clean and neutral canvas.
- Accent color: Warm, contrasting orange (#FFA07A) for buttons and important UI elements.
- Headline font: 'Poppins', a geometric sans-serif (sans-serif), lending a modern and stylish touch.
- Body font: 'Inter', a grotesque-style sans-serif (sans-serif), for readability and a clean interface.
- Use modern, minimalist icons for content types and actions, with subtle color accents to provide a visual guide.
- Clean and structured layout with clear sections for input, content type selection, and output display. Use a card-based design to compartmentalize information effectively.
